# capscr-plugins

Official plugins for [capscr](https://github.com/lintowe/capscr).

## Plugins

| Plugin | Description |
|--------|-------------|
| **borders** | Adds customizable borders and drop shadows to captures |
| **sounds** | Plays custom sounds on capture events |
| **hotbar** | Floating toolbar with Windows acrylic glass effect |

## Installation

Build with `cargo build --release`, then place the compiled libraries in capscr's plugins directory.
